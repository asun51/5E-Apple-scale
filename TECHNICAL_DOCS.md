# 技术原理文档 (给好奇宝宝版)

这份文档用最简单的话解释了这个程序是怎么工作的。

## 1. 它是怎么知道重量的？
你的 MacBook 触控板不仅仅是一块玻璃，它下面藏着**压力传感器 (Force Sensors)**。
*   当你用力按的时候，传感器会感受到压力。
*   平时你“用力按”是为了查字典或者预览文件。
*   这个程序“窃听”了这个压力数据，把它转换成了数字。

## 2. 为什么只能用 Safari 浏览器？
因为只有苹果自家的 **Safari 浏览器** 允许网页读取这个“压力数据”。
Chrome 和 Firefox 浏览器为了安全或者其他原因，把这个功能关掉了，所以它们读不到数据。

## 3. 为什么重量是“估算”的？
因为苹果并没有直接告诉程序“现在是 100克”。
它给程序的是一个 **0 到 1 之间的小数**：
*   0 = 没碰
*   1 = 普通点击的力度
*   2+ = 用力按的力度

我们通过数学公式（猜的），把这个 0-1 的数字变成了克数。所以它可能不准，比如它说 100克，实际可能是 90克或者 110克，但用来玩玩足够了！

## 4. 代码结构 (给想学的人)
这个程序只有三个文件，非常简单：
*   **`index.html` (骨架)**：决定了页面上有哪些东西（标题、数字、按钮）。
*   **`style.css` (皮肤)**：决定了页面长什么样（颜色、字体、圆角）。我们模仿了德国博朗 (Braun) 的经典设计风格。
*   **`script.js` (大脑)**：负责监听触控板的压力，然后更新屏幕上的数字。

## 5. 隐私安全
放心，这个程序**完全运行在你的电脑上**，不需要联网，也不会把你的指纹或者压力数据上传到任何地方。
